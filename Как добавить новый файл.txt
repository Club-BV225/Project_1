Как добавить новый файл?
"Очень просто." - хотелось бы так сказать, пока не могу. Но это должно быть простым для всех нас.
Процедура выглядит так:
Исходное состояние:
  - у Вас есть аккаунт на ГитХаб;
  - у Вас есть желание участвовать в проекте;
  - у Вас есть информация, которую Вы хотите добавить к проекту;
  - у Вас есть ссылка на проект: https://github.com/Club-BV225 ;
  - Вы находитесь в своем аккаунте ГитХаба.
  
 Теперь собственно операция по добавлению нового файла в общий проект:
  - переходите по ссылке на страницу общего проекта;
  - выбираете папку, в которую хотите добавить новый файл (выбираете проект);
  - в верхней части нажимаете "add file"(Добавить новый файл). Откроется страница редактирования файла;
  - даете имя своему файлу в строке "Project_1/Имя_Вашего_файла in main";
  - в поле "Edit new file" пишете или вставляете скопированный текст;
  - снизу в поле "Commit new file" добавляете описание коммита (не обязательно, но желательно);
  - нажимаете кнопку ниже "Commit new file";
  - на открывшейся странице нажмите кнопку "Create pull requests";
  - на вновь открывшейся странице нужно ещё раз нажать кнопку "Create pull requests";
  - теперь открылась страница, на которой уже ничего делать не нужно. Переходите по своим делам, в другие проекты, на другие страницы и прочее;
  - дальнейшие действия зависят только от тех, кто может выполнить слияние веток (пока что это только я).
Вы создали новый файл. Молодцы! Правда. Только этот файл имеется только в Вашей копии проекта.
Теперь нужно создать запрос на вытягивание (create pull reqгest). Типа запрос на включение файла в общий проект.
После этого все могут обсудить этот файл (не знаю как эта процедура происходит, во всяком случае, можно ознакомиться с файлом в общем проекте).
НО! файл ещё не включен в общий проект и на главной странице его не видно. Доступ к файлу можно получить через вкладку "Pull requests".
Там можно ознакомиться с внесенными изменениями. Это вроде сборника черновиков, не утвержденных быть в главной ветке (пока не утвержденных).
Дальше участник, имеющий полномочия на слияние веток (пока что это только я) выполнит "утверждение" вашего файла и включит его в общий проект.
То есть, выполнит слияние веток.
Как я это сделаю? Зайду на вкладку "Pull requests", отмечу галочки и нажму кнопку "merge", а затем еще онду кнопку "Confirm merge".
И вот после этого Ваш файл окажется в общем проекте.

Рекомендую всем пройти эту процедуру. Так быстрее научимся работать на ГитХаб.
